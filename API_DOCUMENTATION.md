# API Documentation For a Movie Recommendation System

## Overview

The Movie Recommendation System API was built using FastAPI.

Swagger/OpenAPI documentation is automatically generated by FastAPI to simplify testing and endpoint documentation.

---

## Base URL

```text
http://127.0.0.1:8000
```

---

## API Endpoints

### GET /

Returns API status message.

Response Example:

```json
{
  "message": "Movie Recommendation API Running"
}
```

---

### GET /api/movies

Fetch all movies from the system.

Response:
- List of available movies

---

### POST /api/movies

Create a new movie.

Parameters:
- movie_id
- title
- genre

Response:
- Newly created movie object

---

### POST /api/movies/{movie_id}/checkout

Checks out a movie.

Path Parameter:
- movie_id

Response:
- Success message
- Updated movie status

Error Response:
- 404 if movie is not found

---
