# Reflection For Assignment 11

## Challenges Faced

One of the biggest challenges I faced in this assignment was understanding how to separate business logic from storage logic using the Repository Pattern.

Initially, it was difficult to understand the relationship between repository interfaces, in-memory implementations, and factories. However, after implementing the structure step-by-step, the architecture became much clearer.

Another challenge was organizing the project structure correctly. Proper folder organization was important to ensure imports and testing worked successfully.

Implementing the in-memory repositories using Python dictionaries also required careful planning to ensure CRUD operations worked correctly.
